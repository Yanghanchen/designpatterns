创建型模式：涉及对象实例化，有：单例模式、工厂模式、生成器模式、原型模式
行为型模式：涉及类和对象交互及分配职责，有：模板模式、中介者模式、迭代器模式、访问者模式、备忘录模式、观察者模式、责任链模式、策略模式、状态模式、命令模式、解释器模式
